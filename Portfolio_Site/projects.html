  • Travel Website

    This is a really useful website if you're traveling to a foreign country. You can look up attractions, find places to eat, check the weather, translate to foreign languages, and get updated covid statistics.

    I learned a lot while creating this website, including how to use Sass, the qs library, and how to find free apis online.
    Travel is a single-page application built using HTML, CSS grid, Sass, JavaScript (jQuery, AJAX), and webpack.
    APIs: openweathermap.com, Google Translate, Covid-19 by API Sports, and Truway Places.

  • Battleship Game

    As my first "real project" when I was a month into coding, this is one of the most rewarding projects I've ever built. When I wrote this game, I had not yet learned object-oriented programming, UIs, or most other software concepts and tools I know now. It was made with just the basics - methods, loops, and if/else logic. Recently, I went back to this project, added a UI, and refactored it to include a MVC architecture.

    The skills I gained from this project are more intangible rather than specific technologies. I learned perseverance; although my code crashed countless times, I kept debugging until it worked. I also learned to reason like a programmer and come up with innovative solutions to software problems.
    Battleship was built using Java with a Swing UI.
